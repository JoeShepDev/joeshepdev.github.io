Clyde (AI Project)

This is my personal AI project named Clyde. What you see now is a working display on my watch showing that he's live. Swiping to the left shows a new page where my own custom feed of self selected subreddits load their posts in. Swiping to the right again shows the initial screen showing the status of Clyde. A lot more features are yet to be implemented those include but are not limited to: health detection (heart rate, temperature, tiredness level, etc), note logging, custom OS module (overwriting the default OS), and many others including upgrading the current pages to have a much more modern and intuitive interface (Clyde Status & Reddit Page).

The Lost Souls (Video Game Project)

The Lost Souls is a 2D side-scrolling game developed in Unity. The idea behind the game is to have the player swap between two characters with various movesets to defeat enemies and find your way back home to earth from a dream. The game is currently in development and is being worked on by just myself at this current time.

Life Dashboard (Productivity Project)

The Life Dashboard is the frontend web client for my personal productivity ecosystem. It displays real-time data pulled from Archstone Nexus — my self-built REST API — including weather, personal finance, and a live countdown to upcoming events. Planned additions include calendar integration, a to-do list, and additional data widgets. Built entirely by myself using vanilla HTML, CSS, and JavaScript.

Archstone Nexus (Backend Project)

Archstone Nexus is a REST API built with Java and Spring Boot that serves as the backend for my Life Dashboard project. It provides endpoints for retrieving real-time weather data, personal finance information, and upcoming event countdowns. Future plans include having Clyde connect to it via API calls to pull up information related to his tasks and schedule. Deployed and self-hosted on a personal VPS.
